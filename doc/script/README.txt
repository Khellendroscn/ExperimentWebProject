科技立项：物理实验数据处理系统/数据处理脚本模块
说明：
    老师可以通过在网页上填写实验信息来设计实验，并为实验编写简单的脚本来计算实验误差，
    以此作为评分依据。
脚本语法：
    ・语句：语句之间使用分号';'作为分隔符
        ・赋值语句：<表达式1> = <表达式2>;
            求出表达式2的值，赋予表达式1
        ・计算误差语句：<表达式> == <计算公式>;
            分别求出表达式（实际值）和计算公式的值（理论值），并计算误差率
            （误差率 = |实际值-理论值|/理论值）
        ・提交语句：submit <表达式>：
            提交计算结果，系统根据提交的值来判断分数，每个脚本必须有且仅有一个submit语句，
            且submit语句必须位于脚本的末尾，在此之后的语句都不会被编译

    ・变量：变量使用$<变量名>来表示，变量有三种数据类型。
        Null:未赋值的变量为Null类型,Null类型的变量可以使用任意类型的变量进行赋值
        Num:单个数值类型（浮点数），可以进行+-*/运算，只能使用Num类型进行赋值
        Array:数组类型，可以进行+（添加元素），[下标]运算，只能使用Array类型进行赋值
            使用{<元素1>[,<元素2>...]}可以创建数组

    ・函数：函数调用语法<函数名>([<参数1>[,<参数2>...]]);
        不能自定义函数，只能调用内建函数：
            Num sum(<一维数组>):
                返回参数数组每一项相加之和
            Num avg(<一维数组>):
                返回参数数组的平均值
            Num sqrt(Num):
                返回参数的平方根
            Num pow(Num<底数>,Num<指数>):
                求乘积函数
            Num abs(Num):
                返回参数的绝对值
            <一维数组> results():
                返回误差结果集（计算误差语句求出的误差都保存在结果集中）
